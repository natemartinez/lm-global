Design Document: Tech Care Dashboard
Overview
A responsive, single-page medical dashboard application prioritizing accessibility, SEO, and performance through lazy loading. The layout strictly follows a three-column architecture wrapped in a main flex container, with CSS Grid specifically applied to the diagnosis report metrics for rigid box constraints.

Layout Architecture (Based on Screenshot_2026-07-10_18-00-39.png)

Header (Flexbox)

Left: Brand container

Center: Navigation menu (Overview, Patients, Schedule, Message, Transactions)

Right: Provider profile and settings

Main Content Layout (Flexbox)

Left Column (Aside): Patient roster list with search functionality. Items use Flexbox for avatar and detail alignment.

Center Column (Main):

Diagnosis History: Contains a placeholder for the main chart.

Vitals/Metrics Container (CSS Grid): A 3-column grid layout containing three distinct cards (Respiratory, Temperature, Heart Rate) to ensure structural integrity across viewport changes.

Diagnostic List: A tabular data structure for condition, description, and status.

Right Column (Aside):

Patient Profile: Focused view of the selected patient's demographic and contact information.

Lab Results: List of downloadable reports.

Constraints Applied

No specific numbers, logos, or graphics: Replaced with generic textual placeholders and CSS shape stand-ins.

Semantic HTML & ADA: Uses <nav>, <main>, <aside>, <article>, <section>, and aria-labels.

Performance: Core dashboard widgets are lazy-loaded via React Suspense.

3. React Application (App.tsx)
TypeScript
import React, { useEffect, useState, Suspense, lazy } from 'react';
import './App.css';
import { DashboardResponse } from './types';

// ADA/Performance Best Practice: Lazy Loading components
const PatientList = lazy(() => import('./components/PatientList'));
const DiagnosisView = lazy(() => import('./components/DiagnosisView'));
const PatientProfile = lazy(() => import('./components/PatientProfile'));

export default function App() {
  const [data, setData] = useState<DashboardResponse | null>(null);
  const [loading, setLoading] = useState<boolean>(true);

  useEffect(() => {
    // Fetching from the single REST API
    fetch('http://localhost:3001/api/dashboard')
      .then((res) => res.json())
      .then((json: DashboardResponse) => {
        setData(json);
        setLoading(false);
      })
      .catch((err) => console.error('Error fetching data:', err));
  }, []);

  if (loading) return <div aria-live="polite">Loading dashboard...</div>;
  if (!data) return <div aria-live="polite">Failed to load data.</div>;

  return (
    <div className="app-container">
      {/* Header */}
      <header className="app-header" role="banner">
        <div className="brand-placeholder"><strong>[Brand Name]</strong></div>
        <nav className="main-nav" aria-label="Main Navigation">
          <ul>
            <li>[Overview]</li>
            <li className="active">[Patients]</li>
            <li>[Schedule]</li>
            <li>[Message]</li>
            <li>[Transactions]</li>
          </ul>
        </nav>
        <div className="user-profile-placeholder">
          <div className="avatar-circle"></div>
          <div>
            <strong>[Provider Name]</strong>
            <br />
            <span>[Title]</span>
          </div>
        </div>
      </header>

      {/* Main Content Layout */}
      <div className="dashboard-layout">
        
        {/* Left Column */}
        <aside className="sidebar-left" aria-label="Patient Directory">
          <Suspense fallback={<div>Loading Patients...</div>}>
            <PatientList patients={data.patients} />
          </Suspense>
        </aside>

        {/* Center Column */}
        <main className="main-content" role="main">
          <Suspense fallback={<div>Loading Diagnosis...</div>}>
            <DiagnosisView history={data.diagnosisHistory} list={data.diagnosticList} />
          </Suspense>
        </main>

        {/* Right Column */}
        <aside className="sidebar-right" aria-label="Patient Profile and Lab Results">
          <Suspense fallback={<div>Loading Profile...</div>}>
            <PatientProfile labResults={data.labResults} />
          </Suspense>
        </aside>

      </div>
    </div>
  );
}
